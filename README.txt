O projeto implementa um SequenceSet baseado em uma base de dados disponibilizada no arquivo CSV que pode ser encontrada nesse diretorio como 'data.csv'.

O sistema pode:
    Importar registros de um arquivo .csv
    Exportar registros do SequenceSet para um arquivo .csv
    Inserir novos registros
    Remover registros especificos
    Buscar registros especificos
    Visualizar registros e blocos no SequenceSet


Recomendamos o uso de alguma IDE ou editor de texto como o VSCode para uma melhor experiencia.

Para executar o programa tenha certeza de ter pelo menos alguma versao do GCC/G++ instalado,
compile o arquivo 'App.cpp',
se for a primeira vez executando o programa garanta que o arquivo contendo os dados (data.csv) esta no mesmo diretorio do executavel gerado, 
para distribuicoes linux ele deve ficar na pasta padrao do projeto, para windows deve estar na pasta output.
